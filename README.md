## 🤔 为什么需要这个 Skill？

你和一个女生聊得正热，她突然说"好困"。你回什么？

❌ **没学过的人**：「那早点休息吧，明天记得吃早饭哦」
→ 越级关心，需求感爆炸。她已经把你划进备胎区。

✅ **Cyrus 教你这样想**：先看信号——前面聊得热，她在分享状态。前面回得冷，她在给你体面离场。策略完全不同。

这就是这个 Skill 做的事：**不是给你一个万能回复模板，而是教会 AI 用 Cyrus 的思维方式拆解每一次聊天场景**。

## 🧠 它怎么想问题

Cyrus 不是话术机器，他有一套完整的判断框架：

```
信号读取 → 态度判断 → 分类对话 → 选择策略 → 执行 & 回撤
```

**核心思维模型：**

- **态度 → 情绪 → 想法**：三步递进，跳过任何一步就翻车
- **分化对待**：好女孩走长期维护，坏女孩走效率路线
- **窗口理论**：窗口出现立刻升级，窗口关闭不再恋战
- **情绪价值公式**：情绪价值 = 情绪收益 − 情绪成本
- **不犯错优先**：宁可什么都不做，也别做错

## 🔄 如何更新这个 Skill

这个 Skill 支持持续进化：

1. 把你新的文章、聊天记录、反馈放到 `references/` 目录
2. 在对话中说"有新的素材，请更新"
3. AI 会读取新素材，提出对 SKILL.md 的修改建议
4. 你确认后自动更新

如果你发现 AI 回答得不像你——直接指出来：
> "你这里不像我，应该是这样……"

这个反馈会被吸收进 SKILL.md。

这个 Skill **不做什么**：
